# Unit Price Growth Control

Интерактивный контрол для визуализации роста стоимости юнита в процентах на протяжении 30 лет. Построен с использованием React, TypeScript, Vite и shadcn/ui.

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик
- **shadcn/ui** - компоненты UI
- **Tailwind CSS** - стилизация
- **Canvas API** - отрисовка графика

## Установка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build

# Превью production сборки
npm run preview
```

## Функции

- ✅ Интерактивный график с drag-and-drop точками
- ✅ Клавиатурное управление (стрелки для навигации и изменения значений)
- ✅ Автоматическое построение графика с заданным процентом роста
- ✅ Динамический диапазон с автоматическим расширением
- ✅ Отображение года сдачи юнита красной линией
- ✅ Проценты изменения над каждой точкой (зеленым/красным)
- ✅ Все UI компоненты на shadcn/ui

## Деплой на Vercel

Проект готов для деплоя на Vercel. Используйте Vercel CLI:

```bash
npm i -g vercel
vercel
```

Или подключите репозиторий к Vercel через веб-интерфейс - Vercel автоматически определит настройки.
